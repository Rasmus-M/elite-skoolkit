AB0B: Multiply MSB
Used by the routines at LA0FD, LA1EA, minus_4_a_b, mult_b_coord_scaled, face_normal_component, LAE8A and mult_2_A_EC.
Calculate A = MSB(B * C) using square lookup table.
Input
B First number
C Second number
Output
A MSB(B * C) (plus 1 in some cases)
mult_b_c_msb AB0B LD A,B A = B
AB0C SUB C A = B - C
AB0D JR NC,mult_b_c_msb_0
AB0F NEG A = ABS(B - C)
mult_b_c_msb_0 AB11 SRL A A = ABS(B - C) / 2
AB13 LD H,$BF H = MSB of square table at BF00
AB15 LD L,A L = ABS(B - C) / 2
AB16 LD D,(HL) D = ((B - C) / 2)^2 / 256
AB17 LD A,B A = B
AB18 ADD A,C A = B + C
AB19 RRA A = (B + C) / 2
AB1A LD L,A L = (B + C) / 2
AB1B LD A,(HL) A = ((B + C) / 2)^2 / 256
AB1C SUB D A = (((B + C) / 2)^2 - ((B - C) / 2)^2) / 256 = B * C / 256
AB1D RET

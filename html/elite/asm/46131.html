B433: Line up steep
Draws a steep line from bottom to top
Self-modified for drawing right or left
Input
HL Screen buffer address
D Length in pixels
E Start slope offset
B Start pixel line 1-8 in character row
C Start pixel value to draw
LB433 B433 DEC H Previous line
This entry point is used by the routine at draw_clipped_line.
